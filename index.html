Smart Water Monitoring

AquaSense is an IoT-based smart water monitoring solution designed to monitor groundwater conditions in real time. The system collects and displays live sensor data through an interactive web dashboard, helping users make informed decisions regarding water usage and conservation.

Water Level Log

Ultrasonic sensors log depth parameters continuously.

Flow Measurement

Hall-effect sensor monitors pump discharge rates.

Purity Indicators

Integrated TDS & turbidity checks water cleanliness.

IoT Cloud Uplink

Syncs telemetry datasets to cloud feeds instantly.

Sensor Array

Hardware Sensor Network

AquaSense utilizes five specialized environmental sensors to record physical, chemical, and thermal dynamics.

Water Level Sensor

Ultrasonic sensor (HC-SR04) mounted inside the top cap calculates depth table drawdowns via sound reflections.

Water Flow Sensor

YF-S201 Hall-effect sensor counts spinner rotations to calculate discharge velocity (L/min) and total cumulative consumption.

TDS Sensor

Measures Total Dissolved Solids in ppm, tracking dissolved minerals and salts to gauge potable suitability.

Turbidity Sensor

Infrared sensor calculates suspended sediment density in NTU, identifying sand/mud intrusions in real time.

Temperature Sensor

Waterproof probe (DS18B20) records aquifer thermal fluctuations to flag pump heating or runoff anomalies.

Key Features

System Capabilites

Real-Time Monitoring

Pulls sensor readings continuously and updates the portal logs at 15-second cycles for immediate data views.

Cloud Connectivity

Utilizes HTTP/MQTT endpoints to push logs securely to the ThingSpeak cloud channel repository.

Data Visualization

Translates complex, raw measurements into clean line/bar trend charts using local Chart.js setups.

Instant Alerts

Generates alerts for critical metrics like low level limits or high sediment turbidity indicators.

Water Conservation

Empowers administrative actions by budgeting pump cycles based on aquifer drawdown rates.

Remote Access

Web portal uses responsive grids, providing access to borewell data from any browser device.

System Pipeline

How It Works

AquaSense operates a pipeline converting raw physical data into screen graphs.

1
Sensor Collection

Five sensors query level, flow, temperature, TDS, and clarity parameters.

2
ESP32 Processing

MCU processes inputs and communicates wirelessly via Wi-Fi.

3
ThingSpeak Cloud

Stores telemetry logs in dedicated online databases.

4
Web Dashboard

Pulls raw logs via AJAX to render charts and panels.

5
User Monitoring

Administrators check logs and receive advisories.
