About AquaSense

Smart Borewell Monitoring System

AquaSense is an IoT-based Smart Borewell Monitoring System developed to monitor groundwater resources in real time. The system integrates ESP32 technology, cloud storage, and multiple water quality sensors to provide continuous monitoring and intelligent decision-making.

Working Principle

System Architecture

A simple multi-layered process pipeline from physical water wells to client browser views.

Data Processing Workflow

The system starts at the Borewell Casing Cap, where the HC-SR04 ultrasonic level sensor is positioned downwards. When triggered by the ESP32 microcontroller, it fires sound waves and records echo flight time. Simultaneously, spinner ticks on the flow sensor are gathered.

Water purity details are recorded using the submerged TDS Probe (calculating conductivity) and the Turbidity Light Sensor. These inputs are aggregated by the ESP32 microcontroller, converted to digital metrics, and pushed to ThingSpeak IoT channel via HTTP JSON requests.

Finally, the client-side JavaScript dashboard polls the ThingSpeak REST API endpoints every 15 seconds to animate circular gauges, redraw line trends, and check thresholds to trigger alerts.

Key Project Benefits

Future Scope

The AquaSense monitoring prototype can be further scaled to include advanced environmental functions:

Solar-Powered Operations

Equip hardware boxes with mini solar arrays and Li-Po charging circuits to sustain operations in off-grid agricultural fields.

B
Automated Valve Controls

Connect solid-state relay shields to the ESP32, allowing remote or automatic shutoffs of water mainlines when contamination is flagged.

Integrate predictive analytics algorithms to forecast borewell levels based on local rainfall cycles and historical drawdowns.
